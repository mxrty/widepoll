# Widepoll

www.widepoll.com

A web framework for representing granular user opinions on user-submitted content.

Technologies used:

- React
- Node.js
- Express
- PostgreSQL
- jest
- react-testing-library
- JWT
- Redux
- Immer
- Typescript
- Slate.js
- Ant Design
- Formik
- Yup
- Axios
- Redux Thunk
- Lodash
- Chart.js
- Heroku
